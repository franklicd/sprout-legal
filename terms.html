Description

Sprout is a productivity application designed to help users manage tasks, build habits, and improve focus. The app is available as a free download with optional Pro features available via in-app purchase.

Free Plan

The free plan includes:

  • Up to 5 active tasks
  • Up to 3 tasks created per day
  • Up to 2 focus sessions per day
  • Up to 2 active habits
  • Basic progress tracking

Pro Plan

Pro is a one-time in-app purchase via the Apple App Store that unlocks:

  • Unlimited tasks
  • Unlimited focus sessions
  • Unlimited habits with history
  • Leaf rewards and collection
  • Full focus session messages
  • AI-powered features (when available)
